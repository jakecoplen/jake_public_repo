# Project Title: bikeshare.py


## Overview
This is a python program that allows the user to explore data from a bikeshare system for 3 cities:
- New York City
- Washington
- Chicago

It displays statistics on the rider data.

## Files Used:
- `chicago.csv`
- `washington.csv`
- `new_york_city.csv`
